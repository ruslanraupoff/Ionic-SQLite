# Ionic SQLite CURD Mobile App

## Install
```shell
git clone https://github.com/ruslanraupov/Ionic-SQLite.git
cd Ionic-SQLite
npm install
ionic cordova platform add android
ionic cordova platform add ios
```

## Run
```shell
ionic cordova run ios
ionic cordova run android
```
